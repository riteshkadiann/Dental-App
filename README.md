## Getting Started

To run our group project locally:

1. **Download all files from the repository** (or clone the repository using Git):
   ```bash
   git clone https://github.com/riteshkadiann/Dental-App.git
   ```

2. **Open the folder in VS Code** and then open the terminal.

3. **Navigate to Client**:
   ```bash
   cd Dental_App/mern_skeleton/client
   ```

4. **Install dependencies**:
   ```bash
   npm install
   ```

5. **Start the development server**:
   ```bash
   npm run dev
   ```

6. **Access the application**: Click the localhost link provided in the terminal to view the website (typically http://localhost:5173).
